Computational Complexity of the Game of SET

Maggie Burkart

The Game of SET

SET is a popular pattern-recognition card game, which can either be played as a competitive speed game or a collaborative effort. The goal is to find "sets" of three cards out of all cards on the board. The symbols on the cards have four attributes: color, number, shape, and shade. Each attribute can have one of three possible values, as seen below:

Attribute Possible Values
Color Red, Green, Purple
Number One, Two, Three
Shape Oval, Diamond, Squiggle
Shade Full, Shaded, Empty

A "set" is that for each attribute have either all the same values or all different values.

The game of SET poses many interesting computational problems, such as finding a singular set, finding the maximum number of sets, or finding the minimum number of sets possible, which is the problem that I intend to discuss.

Problem: MIN 3-VALUE r-SET

The problem that I discuss in this post is the MIN 3-VALUE r-SET problem. Given a set of cards, a single player is trying to remove the smallest possible number of sets so that no more sets are left on the table in r rounds. In other words, a single player is trying to ruin the game for everyone else who wants to find a set!

NP-Hardness Proof

The MIN 3-VALUE r-SET problem is NP-hard. The proof of this depends on:
            (1) a graphical representation of the set board and
            (2) a reduction from INDEPENDENT EDGE-DOMINATING SET.

Graphical Representation of SET

The board can be represented as a graph G (V, E) in which:
            Each card is a vertex
            Each set is an edge (hyper-edge)

The image to the right represents the graph of a SET board.

The blue outlines around gorups of cards represent hyper-edges, or "sets" in the game.

Independent Edge-Dominating Set

An edge dominating set for a graph G = (V,E) is a set of edges M such that all edges in E that are not in M are adjacent to an element of M.

All other sets (not included in M) need one card that is in a set in M. In the SET scenario, once the sets in M are removed, there must be no remaining sets on the board.

An independent edge dominating set is an edge dominating set in which no two elements are adjacent.

In terms of the game, none of the sets in M contain any of the same cards.

INDEPENDENT EDGE DOMINATING SET is known to be NP-hard.

Why does this reduction work?

Picking a set of cards eliminates the edge of that set, the vertices of the included cards, and the edges of possible sets that included one of those cards. To eliminate all other possible sets (the edges in the graph), the sets chosen must belong to an indepedent edge-dominating set.


Specifically, this subset of chosen sets must be the smallest possible.


Therefore, we can see intuitively that the problem looks very similar to MINIMUM INDEPENDENT EDGE-DOMINATING SET.

Formal Proof:

An independent edge dominating set of size at most r in G corresponds to an equal number of sets that overlap all other sets.

We want the minimum independent edge-dominating set (the smallest number of sets that overlap all other sets).

This means that an instance of the MINIMUM INDEPENDENT EDGE-DOMINATING SET can be reduced to an instance of a MIN 3-VALUE r-SET.

We know that MINIMUM INDEPENDENT EDGE-DOMINATING SET is NP-hard. Therefore, MIN 3-VALUE r-SET is NP-hard.

Proof Evaluation

This proof seemed pretty intuitive to me, since a SET board lends itself so well to such a graphical representation. The idea that finding the minimum number of sets possible to destory all other sets is NP-hard also makes sense to me, because it would be necessary to first find all possible sets in the first place. I found it interesting that the paper allowed all cards to be chosen from at once. In a real game, only 12 cards would be visible at a time, and cards would be replaced as sets were found. I would imagine that this would make the problem much harder to solve.
